## 一、项目概述
本项目针对“图书角”Web应用进行UML建模，绘制了用例图与类图，并使用Git进行版本控制管理。

## 二、UML建模工具说明
- **使用工具**：Rational Rose
- **提交形式**： Rational Rose 源文件 `BookCornerModel.mdl`。
- **文件说明**：
  - 用例图位于：`Use Case View` -> `图书角系统用例图`
  - 类图位于：`Logical View` -> `图书角系统类图`

## 三、核心功能设计说明
1. **用例图角色划分**：
   - 用户：图书搜索、借阅图书、归还图书、查看借阅记录。
   - 管理员：图书信息录入、修改/删除图书、查看所有借阅记录。
2. **类图设计**：
   - 包含 `Book`（图书）、`User`（用户）、`BorrowRecord`（借阅记录）三个核心类。
   - 关系：一个用户对应多条借阅记录（1对多），一本图书对应多条借阅记录（1对多）。
